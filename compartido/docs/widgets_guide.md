# Guía de Widgets - Compartido

> Guía de uso de widgets reutilizables

## Introducción

Este módulo proporciona widgets PyQt6 reutilizables para los simuladores y UX del proyecto.

## Widgets Disponibles

### LedIndicator

Indicador LED con estados de color.

### ConfigPanel

Panel de configuración con validación de IP y puerto.

### LogViewer

Visor de logs con formato y colores.

### StatusIndicator

Indicador de estado con texto descriptivo.

### ValidationFeedback

Componente de feedback visual para validación.
