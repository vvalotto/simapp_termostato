# API Reference - Compartido

> Referencia de APIs públicas del módulo compartido

## Networking

### EphemeralSocketClient

Cliente TCP con patrón "conectar → enviar → cerrar" para comunicación efímera.

### BaseSocketClient

Cliente base con soporte asíncrono para comunicación TCP persistente.

### BaseSocketServer

Servidor TCP con threading para manejo de múltiples conexiones.

## Widgets

### LedIndicator

Indicador LED visual (rojo/verde) para estados.

### ConfigPanel

Panel de configuración IP/puerto con validación en tiempo real.

### LogViewer

Visor de logs con colores y formato.

### StatusIndicator

Indicador de estado con texto y colores.

### ValidationFeedback

Feedback visual para validación de formularios.

## Estilos

### ThemeProvider

Proveedor de tema oscuro consistente para toda la aplicación.

### ThemeColors

Constantes de colores del tema.
